# El Pro

给element-plus加载组件

在线访问地址: https://52css.github.io/el-pro/

## 按钮

* 支持防重点击
* 如果是删除按钮，替换成Popconfirm

## curd

* 一次数据定义，适用于表单、查询、详情、列表等场景。

## form

* 支持分栏

## operation

* 当操作按钮超过父长度，自动折叠

## query

* 支持搜索

## status

* 显示状态信息

## table

* 左边、右边固定、固定最小宽度
* 多选
* 单选
* 后端分页
* 前端分页
* 支持拖拽
* 支持标题和toolbar
* 支持固定头部和尾部
